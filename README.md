# HousePricePrediction

This project analyzes and predicts house prices using the Kaggle Housing Prices dataset. It features a complete machine learning workflow, from data cleaning and exploratory data analysis (EDA) to training and comparing predictive models.

By comparing a baseline Linear Regression model against a more complex Random Forest Regressor, this project identifies the key structural features (like square footage and number of bathrooms) that most heavily influence a property's market value.

Tech Stack: Python, Pandas, Scikit-Learn, Matplotlib, Seaborn
